# FileSorting & Data Integration
## Overview
This project provides a smart solution for companies that handle large amounts of unorganized text data. It automatically reads, categorizes, and stores information into a secure database, turning manual data entry into an automated workflow.
* Eliminates the need for manual sorting of text files, saving significant administrative time
* Automatically checks for duplicates to ensure your records remain clean and professional
* Displays processed information through a modern web interface, making data easy to find for all team members

## Key Features
* **Automated Scanning:** The system "reads" through unorganized documents and identifies key details like names and categories
* **Secure Storage:** Information is moved directly into a professional SQL database for long-term safety.
* **Interactive Dashboard:** A web-based portal where you can view current data and system status in real-time.

## Technology Stack & requirements
* **Microsoft .NET 9:** A robust and modern framework that ensures high performance and security.
* **Blazor:** Used to create the interactive and user-friendly web pages.
* **SQL Server:** A industry-standard system used for reliable data management.

### Requirements
- .NET Core 9.0
- ASP.NET Core 9.0
  . Entity Framework Core 9.0
- DotNetEnv 3.1.1
- Microsoft.EntityFrameworkCore.SqlServer 5.0.11
- Swashbuckle.AspNetCore 7.2.0

## How to Get Started
1.  **Preparation:** Place your unorganized text files in the designated folder.
2.  **Processing:** Run the integration tool to scan and sort the data.
3.  **Review:** Open the web dashboard to see your newly organized information.


## Installation
1. Clone the repository:
```sh
# Using SSh 
ssh git@github.com:krigjo25/Api-FileSorting-cs.git

# Using git bash
git clone https://github.com/krigjo25/Api-FileSorting-cs.git

# Using Github Cli
gh repo clone Api-FileSorting-cs
```

2. Navigate to the project directory
```sh
cd console-scconnectivity-py
```
3. Open the project in Visual Studio / Jetbrainz Raider, and install the requirements
